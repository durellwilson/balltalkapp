Create Firestore Index for BallTalk App

Index Information

This page will help you create the required Firestore index for the BallTalk app. The index is needed for the following query in the DawService.getUserProjects function:

const projectsQuery = query( collection(db, PROJECTS_COLLECTION), where('userId', '==', userId), orderBy('updatedAt', 'desc'), orderBy('__name__', 'asc'), limit(limitCount) );

Index Details:

  • Collection Group: projects
  • Fields:
    • userId (Ascending)
    • updatedAt (Descending)
    • __name__ (Ascending)
Create Index in Firebase Console

Manual Steps:

Step 1: Sign in to Firebase Console

Make sure you're signed in to the Firebase Console with an account that has access to the BallTalk project.

Step 2: Create the Index

Click the button above to open the Firebase Console with the index creation form pre-filled. Then click "Create index" to create the index.

Step 3: Wait for the Index to Build

After creating the index, it may take a few minutes for Firebase to build it. You can check the status in the Firebase Console under "Firestore Database" > "Indexes" tab.

Alternative Methods

Using Firebase CLI:

If you have the Firebase CLI installed, you can create the index by adding it to your firestore.indexes.json file and deploying:

// Add this to firestore.indexes.json { "collectionGroup": "projects", "queryScope": "COLLECTION", "fields": [ { "fieldPath": "userId", "order": "ASCENDING" }, { "fieldPath": "updatedAt", "order": "DESCENDING" }, { "fieldPath": "__name__", "order": "ASCENDING" } ] } // Then run: firebase deploy --only firestore:indexes

Using the Node.js Script:

You can also use the provided Node.js script to create the index programmatically:

// Install Firebase Admin SDK if not already installed npm install firebase-admin // Set up service account credentials export GOOGLE_APPLICATION_CREDENTIALS=/path/to/service-account.json // Run the script node scripts/create-index.js
